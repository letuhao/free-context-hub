## Sweep runs (T2-T5, one task per half-life)

For each N in `[3, 7, 14, 30]`:

**T2.N.1 — Reset lesson_access_log to T0.**
- Cmd:
  ```bash
  T0=$(cat ./.scratch/t0.txt | tr -d ' \n')
  docker compose exec -T db psql -U contexthub -d contexthub -c \
    "DELETE FROM lesson_access_log WHERE accessed_at > '${T0}';"
  ```
- Verify: `SELECT COUNT(*)` matches `T0_ROW_COUNT` (recorded in T0.2).
- If count diverges by more than 1 row (could be race with bootstrap), inspect.

**T2.N.2 — Recreate mcp with new half-life env.**
- Cmd: `LESSONS_SALIENCE_HALF_LIFE_DAYS=<N> docker compose up -d --force-recreate mcp`
- Wait for ready (poll API endpoint).
- Verify env: `docker compose exec -T mcp node -e "console.log(process.env.LESSONS_SALIENCE_HALF_LIFE_DAYS)"` → `<N>`.

**T2.N.3 — Run baseline.**
- Cmd: `npm run qc:baseline -- --tag sprint-12.1e2-hl<N> --samples 3 --surfaces lessons 2>&1 | tee ./.scratch/run-hl<N>.log`
- Expected runtime: ~15 min (40q × 3 samples × 1 run; hardware-dependent).
- Verify: `ls docs/qc/baselines/2026-04-19-sprint-12.1e2-hl<N>.{json,md}` → both exist.
- Verify: `node -e "const d=JSON.parse(require('fs').readFileSync('docs/qc/baselines/2026-04-19-sprint-12.1e2-hl<N>.json','utf8')); console.log('per_query:', d.surfaces.lessons.per_query.length, 'errors:', d.surfaces.lessons.errors);"` → 40 / 0.

Total: 4 runs × ~15min = ~1 hour wall-clock.
