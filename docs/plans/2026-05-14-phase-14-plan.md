### Task 4 — Write `src/scripts/reembedAll.ts`

**File:** `src/scripts/reembedAll.ts` (NEW)

**Intent:** Implement v3.1 design from `docs/specs/2026-05-14-phase-14-design.md`. Bulk in-place re-embed for 3 tables.

**Key shape requirements:**
- Imports: `fs.promises`, `embedTexts`, `getDbPool`, `bumpProjectCacheVersion`, `getEnv`, `createModuleLogger`
- TABLE_SPECS for `chunks`, `lessons`, `document_chunks` with idColumn + selectCols + buildEmbedText
- CLI args parser: `--project-id`, `--table`, `--batch-size`, `--dry-run`, `--limit`, `--from-id`, `--yes`
- Pre-flight: confirmation prompt unless `--yes`, dim probe (call embedTexts(['probe']) and assert length matches EMBEDDINGS_DIM)
- SIGINT/SIGTERM handler: flush in-flight failed IDs + bump caches before exit 130
- `reembedTable(spec, opts, projectsSeen)`:
  - Keyset pagination: `SELECT WHERE id > $lastId ORDER BY id LIMIT batchSize`
  - Per-batch BEGIN/COMMIT
  - Failed-batch isolation: log + advance keyset + continue
  - BEGIN/COMMIT/pool errors: log + break out of table loop
  - finally{} writes failed IDs to `.phase-gates/failed-<table>-<ts>.json`
  - Per-table cache bump at end (inside reembedTable)
