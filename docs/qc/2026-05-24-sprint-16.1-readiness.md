## Key decisions captured during sprint

1. **Drafter pattern:** Sonnet subagent (no committed drafter code) — saved ~45 min of build time + zero archived ephemeral code. Per the no-rush + design-first principles.
2. **Refined drafter prompt (v2):** 7-rule explicit ruleset added after pilot v1 showed compound facts, verbatim-string facts, length drift, and answer-facts asymmetry. Yielded 9/10 quality on re-pilot.
3. **`no_answer` validator policy:** distribution check counts only `drafted_by='human'` rows, not bootstrap `no_answer` rows from adversarial-miss groups (those pre-date Phase 16 and shouldn't be conflated with edge cases).
4. **Query count correction:** mid-sprint discovery that `grep -c '"id"'` over-counts due to nested `target_any` ids. True bootstrap count is 127 (not 138). All docs updated.
5. **Pre-commit hook deferred (A5):** project has no existing husky/.githooks; adding for one ephemeral sprint = avoidable npm dependency.
