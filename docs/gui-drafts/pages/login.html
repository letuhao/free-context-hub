State 1 — Login (password)

C
ContextHub

Sign in

Operator access to the governance console.

Forgot?

No account? Registration is by invite only.

State 2 — MFA challenge (reaches AAL2)

Two-factor

Enter the 6-digit code from your authenticator app.

State 3 — Soft lockout (rate-limit / anti-automation)

Too many attempts

Sign-in is temporarily paused for this account. Try again in 4:38.

Soft lock — an ever-increasing delay after repeated failures (OWASP V6, ≤100 fails/hr). It does not reset your hard-lock status and cannot be triggered against you by a password-reset attempt.

State 4 — Forgot password (never locks the account)

Reset password

We'll email a single-use, short-lived link. For security, the response is the same whether or not the address has an account.

If an account exists for that address, a reset link is on its way.

State 5 — Enforcement OFF (no login required)

Single-operator mode

MCP_AUTH_ENABLED=false — the console is open as the root/dev operator; no login screen is shown.

Enable enforcement and configure a root credential before exposing this deployment to any untrusted network.
