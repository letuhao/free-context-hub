Project Summary
free-context-hub is a self-hosted persistent memory and guardrails system for AI agents. It provides MCP-compatible tools for managing lessons, guardrails, code search, and generated documentation. The backend is built with Node.js and uses SQLite for storage. The GUI is a Next.js dashboard that connects to the API server. Key features include tiered code search (exact, glob, semantic), project groups for multi-repo knowledge sharing, a background job queue with dead-letter support, and AI-generated documentation (FAQ, RAPTOR summaries, QC reports).
