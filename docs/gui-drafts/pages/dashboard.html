About this project

free-context-hub is a self-hosted persistent memory and guardrails system for AI agents. It provides MCP-compatible tools for managing lessons, guardrails, code search, and generated documentation. The project uses a Node.js backend with SQLite storage and a Next.js GUI dashboard.
