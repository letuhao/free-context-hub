Use retry with exponential backoff for external API calls

decision active Press ESC to close

Content

When calling external APIs (LLM providers, GitHub, remote git), always wrap the call in a retry loop with exponential backoff. This prevents transient network errors from causing job failures.

Recommended Pattern

Use the retry() helper from src/lib/retry.ts:

typescript
const result = await retry(
() => fetchFromProvider(params),
{ maxAttempts: 3, backoff: 'exponential' }
);

Rules

  • Max 3 retries. Initial delay 500ms, doubling each time.
  • Log each retry attempt at warn level.
  • Only retry on transient errors: ECONNRESET, ETIMEDOUT, RATE_LIMITED.
