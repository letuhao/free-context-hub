State 0 — First run detected (no root configured)

Set up the trust anchor

This deployment has no root yet. The root is the out-of-band trust anchor — only the holder of DATABASE_URL can establish it. You are reaching this page from localhost with enforcement off, which is the install-time window.

To confirm you hold the deployment secret, paste the value of ROOT_BOOTSTRAP_TOKEN (printed once in the server logs at first start). This proves out-of-band possession — it is not a password and is consumed on use.

Prefer the CLI? npm run bootstrap:root does the same, headless. Either path seeds exactly one root principal; running it again is a no-op once root exists.
