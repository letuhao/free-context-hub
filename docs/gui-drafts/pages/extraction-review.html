3. Retry Strategy

All external API calls must implement exponential backoff with a maximum of 3 retry attempts. The base delay is 1 second with a multiplier of 2.

3.1 Configuration

Parameter Default Description
maxRetries 3 Maximum number of attempts
baseDelay 1000ms Initial delay before first retry
multiplier 2.0 Exponential backoff multiplier
