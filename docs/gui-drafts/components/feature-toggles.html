Feature Toggles

Section in /projects/settings. Toggles stored in projects.settings JSONB. Wired to PUT /api/projects/:id.

Features Section (in project settings page)

Features

Saved to project settings

Enable or disable features for this project. Changes take effect immediately.

Git Intelligence
Auto-ingest commits, suggest lessons from git history
Knowledge Graph
Neo4j symbol extraction and dependency tracing
Requires NEO4J_URI to be configured
AI Distillation
LLM-powered reflection, compression, and project summaries
Auto Review
Route AI-generated lessons to review inbox based on agent trust level
Search Reranking
Cross-encoder reranking for more accurate search results

Data model (projects.settings JSONB)

{
  "features": {
    "git_ingest": true,
    "knowledge_graph": false,
    "distillation": true,
    "auto_review": true,
    "reranking": false
  }
}

// Read: GET /api/projects → project.settings.features
// Write: PUT /api/projects/:id { settings: {...} }
// Toggle saves immediately via PUT on click
