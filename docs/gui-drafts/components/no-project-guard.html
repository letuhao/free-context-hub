No Project Guard

Shown on data pages when no project is selected or projectId is empty/invalid. Wraps page content.

Variant 1: No project selected (dropdown is empty)

No project selected
Select a project from the sidebar or create a new one to get started.

Variant 2: Project not found (invalid projectId in URL/storage)

Project not found
The project "old-deleted-project" no longer exists.
Select another project from the sidebar.

Usage pattern: wrap page content

{`// In any data page:
export default function LessonsPage() {
  const { projectId } = useProject();

  // Guard: if no valid project, show prompt
  if (!projectId || projectId === "default") {
    return ;
  }

  // Normal page content...
  return 
...
;
}`}
